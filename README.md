# AI Placement Copilot

AI-powered placement preparation platform for students.

## Features
- User Login & Registration
- JWT Authentication
- DSA Progress Tracker
- Aptitude Test Module
- Resume Checker
- Job Recommendation Portal
- Responsive Dashboard UI

## 🛠️ Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB
- JWT
- CSS

##  Purpose
Helps students prepare for placements with smart tools in one platform.
